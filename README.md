# provisionnement-non-vie--m-thode-lognormal-de-christophide
Provisionnement Non-Vie : Modèle de Christofides  Implémentation R du modèle stochastique de Christofides (loi log-normale) pour l'estimation des provisions IBNR. Ce projet modélise la distribution des réserves et calcule les indicateurs de risque (VaR, RMSE) pour le pilotage Solvabilité II.  #Actuariat #IBNR #R
